# Team Task Manager

A beginner-friendly full-stack Team Task Manager application built using the MERN stack.

This application allows users to:

- Signup and Login
- Manage tasks
- Use role-based access (Admin / Member)
- Track task progress
- Store data using MongoDB
- Access the app live online

# Installation Steps

1. Clone Repository
Bash
git clone https://github.com/UMASRI-02/team-task-managerr.git

2. Frontend Setup
Bash
cd client
npm install
npm run dev

3. Backend Setup
Bash
cd server
npm install
npm start
